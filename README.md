# 🎮 坦克大战 - Tank Battle

一个用纯 HTML5 Canvas 和 JavaScript 实现的经典坦克大战游戏。

## 🕹️ 玩法介绍

- **目标**: 消灭所有敌方坦克，保护你的基地（鹰标）
- 玩家控制绿色坦克，在迷宫地图中与红色敌方坦克对战
- 摧毁砖墙可以开辟新路径，但钢墙无法被摧毁
- 水域会减慢坦克移动速度
- 每关敌人数量递增，挑战逐渐升级

## 🎯 操作方式

| 按键 | 功能 |
|------|------|
| `W` / `↑` | 向上移动 |
| `S` / `↓` | 向下移动 |
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `空格键` | 发射子弹 |
| `P` | 暂停/继续 |
| `R` | 游戏结束后重新开始 |

## ✨ 特性

- 🎨 经典像素风格画面，带发光特效
- 🧱 可破坏的砖墙 和 不可破坏的钢墙
- 🌊 水域地形（减速通过）
- 🦅 基地保护机制（基地被毁 = 游戏结束）
- 🤖 智能敌方 AI（随机移动 + 追踪玩家）
- 💥 粒子爆炸特效
- 📈 3个不同关卡，难度递增
- ❤️ 3条生命，死后短暂无敌
- ⏸️ 暂停功能

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可游玩！

也可以使用任意静态文件服务器：

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .
```

然后访问 `http://localhost:8000`
